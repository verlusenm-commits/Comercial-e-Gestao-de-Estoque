# Camisaria Pro – Dashboard Comercial e Gestão de Estoque Inteligente

Este projeto é um **Dashboard Administrativo** robusto e moderno para gestão de lojas de vestuário, desenvolvido com **HTML5, CSS3 e JavaScript**. O objetivo principal é demonstrar o domínio de arquitetura front-end, layouts complexos com CSS Grid/Flexbox e a criação de interfaces ricas em feedback visual para tomada de decisão.

O projeto foi construído para compor meu portfólio e apresentar práticas sólidas de desenvolvimento para vagas de **Estágio em Desenvolvimento Fullstack**.

## ✨ Funcionalidades

* **Painel de Vendas Dinâmico:** Interface para registro de pedidos com seleção de produto, cor e tamanho, incluindo cálculos automáticos de total e manipulação de carrinho em tempo real.
* **Gestão de Estoque com Monitoramento Visual:** Sistema de cards que utiliza lógica de cores (Status OK, Alerta e Crítico) para indicar a necessidade de reposição de mercadorias.
* **Logística Reversa (Trocas):** Módulo dedicado para registro de trocas e devoluções com controle de status e motivos.
* **Design Responsivo e Reflow:** Layout adaptável que se ajusta de forma fluida a diferentes resoluções de tela, garantindo usabilidade em tablets e desktops.
* **UX Empresarial:** Uso de ícones vetoriais (Lucide) e componentes de interface consistentes para uma experiência de software profissional.

## 🛠️ Tecnologias Utilizadas

* **HTML5** – Estruturação de componentes complexos, navegação em abas e formulários.
* **CSS3** – Uso avançado de Grid Layout, Flexbox, variáveis nativas e Media Queries.
* **JavaScript (ES6)** – Manipulação intensiva do DOM, lógica de estados de navegação e tratamento de cálculos financeiros.
* **Git e GitHub** – Controle de versão, histórico de branches e implantação via GitHub Pages.

## 📂 Estrutura do Projeto

```text
├── dashboard.html   # Estrutura principal com arquitetura de seções
├── index.html       # Tela de acesso/login (Portal)
├── css/
│   ├── dashboard.css # Design do painel e lógica visual de status
│   └── style.css     # Estilos globais e resets
└── js/
    └── scripts.js    # Lógica de vendas, estoque e navegação
```

## 🎯 Objetivo do Projeto

Este projeto demonstra competências fundamentais para um desenvolvedor em início de carreira:
* **Arquitetura de Software Front-end:** Organização de múltiplos módulos em uma única página (SPA-like navigation).
* **Lógica de Negócio:** Implementação de regras de inventário e processos de venda.
* **Visual Management:** Aplicação de conceitos de design para facilitar a interpretação de dados críticos.
* **Prontidão para o Fullstack:** Código estruturado de forma a facilitar a futura integração com APIs REST e bancos de dados como SQLite.
