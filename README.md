# Comercial-e-Gestao-de-Estoque
Este projeto é um Dashboard Administrativo robusto e moderno para gestão de lojas de vestuário, desenvolvido com HTML5, CSS3 e JavaScript. O objetivo principal é demonstrar o domínio de arquitetura front-end, layouts complexos com CSS Grid/Flexbox e a criação de interfaces ricas em feedback visual para tomada de decisão.
